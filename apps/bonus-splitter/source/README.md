# bonus-splitter source

`bonus-splitter` の編集可能なNext.jsソースです。公開URLは次の場所です。

https://kobasho-unihota.github.io/myportfolio/apps/bonus-splitter/

## 開発

```bash
npm ci
npm run dev
```

## 静的出力

```bash
npm run build
```

`out/` の内容を一つ上の `bonus-splitter/` 直下へ反映して公開します。既定のbase pathは `/myportfolio/apps/bonus-splitter` です。
