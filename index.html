Works

これまでの解決事例

楽天ペイ → Zaim 自動連携ツール

自動化

課題

手動入力が煩雑で、家計簿入力に毎週30分以上の時間が取られていた。

成果

入力時間をほぼゼロに削減。入力漏れも解消。

対応内容

GASによるメール解析・自動取得を行い、Zaim APIへ連携。iPhoneショートカットからも安定動作するように通信方式を独自に改善。

使用技術

Google Apps Script Fetch API Zaim API
GitHubでコードを見る

Slack打刻 → スプレッドシート勤怠管理

Webアプリ

課題

月末にまとめて勤怠を入力しており、正確性が低く集計作業も負担だった。

成果

Slackスタンプでの打刻を実現し、集計コストを月5時間削減。

対応内容

SlackのEvent APIを利用し、特定のスタンプが押された時刻をGAS経由でスプレッドシートに記録。月末にPDF形式の請求書を自動生成する機能も追加。

使用技術

GAS Slack API Spreadsheet

競合ECサイト価格調査ツール

スクレイピング

課題

毎日3サイトを手動で巡回し価格を確認していたが、見落としが発生していた。

成果

毎朝8時に一覧をチャット通知。価格変更時の即時検知が可能に。

対応内容

Python (Selenium/BeautifulSoup) を定期実行し、対象サイトから価格情報を抽出。前回取得分との差分がある場合のみLINE Notifyで通知。

使用技術

Python Selenium LINE Notify API

Process

安心してお任せいただくために

1. ヒアリング・要件整理

まずは現状の課題を詳しく伺います。専門用語は使わず、解決したいことを明確にします。ヒアリング後、要件整理メモを共有し認識齟齬を防ぎます。

2. 提案・仕様合意

実現可能な解決策と見積もりを提示します。簡易的な仕様書を作成し、完成イメージを合意した上で着手します。

3. 実装・進捗共有

開発を行います。副業での稼働となりますが、週に1回は必ず進捗状況をご報告し、ブラックボックス化を防ぎます。

4. 納品・アフターフォロー

動作確認後、納品となります。納品後1週間以内の軽微な修正（不具合対応など）は無償で対応いたします。マニュアル作成も可能です。
